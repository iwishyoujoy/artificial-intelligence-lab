Main info

The theme of knowlege base is poker. We have 5 players except you (each has 2 random cards in a hand) and a table with 5 random cards. In poker we have these combinations:

  • Pair
  • Two Pairs
  • Set
  • Straight
  • Flush
  • Full House
  • Four of a kind
  • Straight Flush
  • Royal Flush

Our Prolog program will check if there are any combinations in anyones's hand or on a table and show you all cards.
